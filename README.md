# 🌐 Ayush Tanty's Portfolio

A modern, responsive, and full-stack personal portfolio website built with **React.js**, **Node.js/Express**, and **MongoDB**.

---

## 🚀 Features

- ⚛️ **Modern Frontend**: Built with React (Component-based architecture)
- 🖥️ **Full-Stack Integration**: Projects and Skills served dynamically via a REST API
- 📱 **Fully Responsive**: Optimized for mobile, tablet, and desktop
- 🎨 **Clean UI**: Minimalist and premium design with glassmorphism elements
- 🧭 **Smooth Scrolling**: Intuitive navigation experience
- 💼 **Dynamic Showcase**: Manage your projects and skills via MongoDB
- 📧 **Direct Contact**: One-click email and social media integration

---

## 🛠️ Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- CSS3 (Vanilla)
- Google Fonts

### Backend
- Node.js & Express.js
- MongoDB (Mongoose ODM)
- Dotenv for configuration
- CORS for cross-origin requests

## 💻 How to Run

### 1. Backend Setup
1. Open a terminal in the `backend/` folder:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure your `.env` file (update your Mongo URI).
4. Start the server:
   ```bash
   npm run dev
   ```

### 2. Frontend Setup
1. Open a new terminal in the root folder:
   ```bash
   npm install
   ```
2. Start the React development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:5173` (or the port shown by Vite) in your browser.
