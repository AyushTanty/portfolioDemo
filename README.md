## 💻 How to Run

1. Clone the repository  
```bash
git clone https://github.com/your-username/portfolioDemo.git
````

2. Open the folder

```bash
cd portfolioDemo
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```

5. Open in browser

```
http://localhost:3000
```
